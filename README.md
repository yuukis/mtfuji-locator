# 富士山ロケータ
Mount Fuji Locator App for Android

## これは何？
* 現在地から富士山の方向・距離を画面に表示する Androidアプリケーション
* スマートグラス[Vuzix Blade](http://www.vuzix.jp/smartglasses/blade.html)用のデモアプリとして作成
* Android 5.0 以上の端末で動作します
